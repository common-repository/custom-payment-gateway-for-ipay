== Description ==
Extends WooCommerce to Process Payments with iPAY Gateway. My Custom Payment Gateway for iPAY extends the functionality of WooCommerce to adding an option during checkout by iPAY. This payment gateway facilitate method of Mobile MPESA, Airtel, Equity(eazzypay) and Credit Card Channel.

== Installation ==

1. Install the plugin like you always install plugins, either by uploading it via FTP or by using the "Add Plugin" function of WordPress.
2. Activate the plugin at the plugin administration page
3. If you want: Open the plugin configuration page, which is located under Settings -> Wocommerce and checkout enable it and then configure iPAY details.
